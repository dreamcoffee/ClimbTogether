# 🧗‍♂️ Climbing Community Project
> **클라이밍 입문자부터 전문가까지, 운동 기록과 소통을 위한 커뮤니티 플랫폼**

---

## 1. 📝 프로젝트 소개
클라이밍을 즐기는 사용자들이 자신의 운동 기록을 남기고, 커뮤니티를 통해 정보를 공유할 수 있는 웹 서비스입니다. 단순한 게시판을 넘어, 개인의 등반 성취를 기록하는 데 중점을 두었습니다.

## 2. ✨ 주요 기능
* **인증 및 사용자 관리**: 회원가입 및 로그인 기능을 통한 개인화 서비스 제공
* **커뮤니티 (CRUD)**: 게시글 작성/수정/삭제
* **마이페이지**: 내 정보 확인 및 내가 쓴 글 관리
* **운동 기록**: 당일 수행한 클라이밍 활동(난이도, 횟수 등)을 기록하는 로그 기능

## 3. 🛠 Tech Stack
| **Language** | Java 17 |
| **Framework** | Spring Boot, MyBatis |
| **Database** | MySQL |
| **Frontend** | React, CSS Modules |

## 🧐 회고 및 아쉬운 점 (Retrospective)

### 🔴 운동 기록 데이터의 활용 미흡
현재는 운동 기록을 단순히 저장하는 단계에 머물러 있습니다. 원래 기획 단계에서는 **기록된 난이도 데이터를 바탕으로 사용자의 등반 레벨을 프로필에 표시**하고, 이를 게시글 작성 시 노출하여 정보의 신뢰도를 높이려 했습니다. 

* **보완 계획**: 기록 조작 방지를 위해 **인스타그램 링크 연동** 기능을 추가하여 실제 등반 영상이나 활동을 증명할 수 있는 장치를 마련할 예정입니다.

### 🔴 커뮤니티 확장성 (크루 및 구인 기능)
시간 관계상 클라이밍의 핵심 재미 요소인 '같이 하기' 기능을 구현하지 못해 아쉬움이 남습니다.

* **추가하고 싶은 기능**: 
    * 함께 등반할 사람을 찾는 **'번개 모임'** 게시판
    * 지속적인 활동을 위한 **'크루(Crew) 생성 및 관리'** 시스템
    * 크루원 모집 및 승인 프로세스
